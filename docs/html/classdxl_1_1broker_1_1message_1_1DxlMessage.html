Detailed Description

Represents a DXL message.

This is a wrapper around an underlying "dxl_message_t" structure. This class provides convenience methods for setting and receiving payloads, as well as simplifying publishing and receiving events via the "DxlMessageService".

◆ getNextBrokerGuids()

const unordered_set< string > * DxlMessage::getNextBrokerGuids ( )

Returns the guids for the next brokers in the path to the destination brokers.

The "next" broker guids are the guids to route the message to from this broker's perspective (the next hop on the way to the final destination brokers).

Returns
The guids for the next brokers in the path to the destination brokers. A "NULL" value indicates that the message is being sent everywhere (unrestricted). An "empty" set indicates that destinations have been specified, but there are no valid "next" hops from this broker's perspective. Therefore, the message should not be routed.
