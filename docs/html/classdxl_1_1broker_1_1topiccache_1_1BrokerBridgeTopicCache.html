bool *  result

Returns true if a subscriber exists for the topic.

NOTE: The result is only valid if the return value from this method is true. The issue is that the cache can become enabled/disabled at various times. Therefore, the return value must be checked to ensure the result is correct. If the return value is not valid, a non-cached form of subscriber lookup must be utilized.
