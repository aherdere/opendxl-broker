111 /*
112  * Function: mosquitto_topic_matches_sub
113  *
114  * Check whether a topic matches a subscription.
115  *
116  * For example:
117  *
118  * foo/bar would match the subscription foo/# or +/bar
119  * non/matching would not match the subscription non/+/+
120  *
121  * Parameters:
122  * sub - subscription string to check topic against.
123  * topic - topic to check.
124  * result - bool pointer to hold result. Will be set to true if the topic
125  * matches the subscription.
126  *
127  * Returns:
128  * MOSQ_ERR_SUCCESS - on success
129  * MOSQ_ERR_INVAL - if the input parameters were invalid.
130  * MOSQ_ERR_NOMEM - if an out of memory condition occurred.
131  */
